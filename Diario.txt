1. Implementacao:
• Adicionado dicionário clientes_conectados para guardar o socket
  de cada usuário autenticado, permitindo entrega de mensagens privadas.
• Implementada função processar_comando() que trata os comandos
  /msg, /usuarios, /ajuda e /sair.
• Modificado listen_multiclient para detectar comandos (mensagens
  que começam com /).
• Adicionada thread de recepção assíncrona no cliente para receber
  mensagens privadas a qualquer momento.

2. Dificuldades:
• ModuleNotFoundError ao rodar o cliente. O Python não encontrava
  o módulo servidor porque o script era executado de dentro de
  Cliente/App/ em vez da raiz do projeto.
• O import do servidor dentro do cliente foi removido pois o cliente
  não precisa da classe do servidor pois usa socket diretamente.
